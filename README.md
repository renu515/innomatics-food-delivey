# Food Delivery Data Analysis Hackathon

## Objective
To integrate transactional, user, and restaurant data from multiple formats
(CSV, JSON, SQL) and perform business analysis to derive insights on order
trends, user behavior, and revenue distribution.

## Datasets Used
- orders.csv – Transactional order data
- users.json – User master data
- restaurants.sql – Restaurant master data

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- SQLite
- Jupyter Notebook

## Final Dataset
final_food_delivery_dataset.csv acts as the single source of truth for
all analysis and MCQ/numerical answers.

## Key Insights
- Gold members have higher average order value
- Revenue is concentrated in a few major cities
- Certain cuisines outperform others consistently
- Seasonal trends are visible in order volume
